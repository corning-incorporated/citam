Scheduler class used to create a typical schedules for an office

class citam.engine.schedulers.office_scheduler.OfficeScheduler(facility, total_timesteps, timestep=1, scheduling_rules=None, meeting_policy=None, entry_exit_window=300, preassigned_offices=None, shifts=None)[source]¶

Bases: citam.engine.schedulers.scheduler.Scheduler

Scheduler class used to create a typical schedules for an office building. This assumes each agent has an office space, go to meetings throughout the day and spend time in special spaces called labs. Supports the use of restrooms and dining areas by agents.
