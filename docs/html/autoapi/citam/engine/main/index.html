Ingest raw floorplan and data files for a given floor of a facility.

This method is executed with the CLI call citam engine ingest

Parameters

  • svg – Path to the floorplan file in SVG format

  • csv – Path to the floorplan metadata file in CSV format

  • facility – Name of this facility

  • scale – The scale of the drawing (default = 1/12)

  • floor – Name of this floor (default=0)

  • output_directory – Custom location to store output (default=cache)

  • buildings – List of buildings to process(default to all buildings found in SVG file)

Raises

FileNotFoundError – If input files are not found

citam.engine.main.find_and_save_potential_one_way_aisles(facility: str, floor: str, outputfile: str, floorplan_directory: str = None, **kwargs)[source]¶

Iterate over nodes and edges in the navigation network and identify segments that could potentially be assigned one-way traffic.

Creates a list of potential one-way nav segments where each segment is a list of successive nodes belonging to that nav segment. Save the list for later retrieval and outputs it to an SVG file for the user.

Parameters

  • facility – Name of this facility

  • floor – Name of this floor

  • outputfile – Location to store output

  • floorplan_directory – Custom directory for floorplans (default=cache)

Raises

  • NotADirectoryError – floorplan_directory is invalid

  • FileMotFoundError – could not find floorplan file

  • FileMotFoundError – could not find navnet file
