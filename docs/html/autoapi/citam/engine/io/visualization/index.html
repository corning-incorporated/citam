Save a snapshot of a simulation with the floorplan, agent positions and number of contacts, doors and any markers to an SVG file.

Parameters

  • walls (List[Path]) – All the walls in the floorplan.

  • agent_positions_and_contacts (List[Tuple[int, int, int]]) – positions and number of contacts for each agent.

  • svg_file (Union[str, pathlib.Path]) – Location of the output SVG file.

  • marker_locations (List[Tuple[int, int, int]], optional) – xy positions of any markers, defaults to []

  • marker_type (int, optional) – the type of marker to use, defaults to None

  • arrows (List[Tuple[int, int], Tuple[int, int]], optional) – list of heads and tails for any arrow, defaults to []

  • doors (List[Path], optional) – list of door elements, defaults to []

  • max_contacts (int, optional) – The maximum number of contacts to use for the colorbar, defaults to 100

  • current_time (int, optional) – the time to show on the clock, defaults to None

  • show_colobar (bool, optional) – whether to show a colorbar or not, defaults to False

  • viewbox (Tuple[float, ...], optional) – the viewbox of the SVG, defaults to None
