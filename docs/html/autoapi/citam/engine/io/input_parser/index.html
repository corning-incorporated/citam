citam.engine.io.input_parser.parse_input_file(input_file: Union[str, pathlib.Path]) → Dict[str, Any][source]¶

Read primary simulation input file in json format, validate values, load floorplans and returns dictionary of model inputs.

Parameters

input_file (Union[str, pathlib.Path]) – path to input file expected in json format

Raises

  • ValueError – If unable to decode JSON file.

  • FileNotFoundError – If the file is not found.

  • MissingInputError – If a required value is not found

  • TypeError – If incorrect value type is found for one or more inputs.

  • ValueError – If an input value is invalid.

  • FileNotFoundError – If path to another to a meetings policy is file is not valid.

  • FileNotFoundError – If path to another to a scheduling policy is file is not valid.

Returns

dictionary of inputs
